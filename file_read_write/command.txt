Git command line basics..

To check the git version:
	$ git --version

To pull 
	$ git pull <URL> <branch name> 				ie: git pull https://.... master
	$ git pull origin <branch name>				ie. git pull origin master
git push

	$ git remode add origin <url>				first time only

	$ git add .
	$ git status							to check the changes
	$ git commit -m "message"
	$ git branch -M <branch name> 			ie: git branch -M main
	$ git git push -u origin <branch name> 		ie: git push -u origin main
